ASP.NET 응용 프로그램

축하합니다! 프로젝트를 만들었습니다.

이 응용 프로그램은 다음으로 구성됩니다.

  • Web API에 대해 설명하는 도움말 페이지
  • Bootstrap을 사용한 테마 설정
  • 인증(선택된 경우 등록 및 로그인 방법 표시)
  • NuGet을 사용하여 관리하는 ASP.NET 기능

응용 프로그램 사용자 지정

  • ASP.NET Web API를 사용하여 HTTP 서비스 시작
  • 사이트 테마 변경
  • NuGet을 사용하여 더 많은 라이브러리 추가
  • 인증 구성
  • 모델에서 ASP.NET Web API 스캐폴드
  • Web API 보안 설정
  • 다른 장치의 Web API 액세스
  • 테스트 및 디버깅용 추적 사용
  • ASP.NET SignalR로 실시간 웹 추가
  • Scaffolding을 사용하여 구성 요소 추가
  • 프로젝트 공유
